Welcome to IPoo server

There are two ways to use IPoo:

  1. Using this page,

  2. Using the Greasemonkey script.

Using this page

On the bottom of this page, you can find the IPoo icon («  »). Clicking on this icon will display IPoo panel. This panel contains an area where you can put an IP address or a hostname. IPoo will display any results it knows about.

Using the Greasemonkey script

Greasemonkey is a Firefox add-on that allows to execute a script to modify a given set of pages. IPoo provides a Greasemonkey script allowing to get information for IP and hostnames displayed on a web page.
